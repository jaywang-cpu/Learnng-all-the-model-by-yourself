
## definition
对参数估计准确性的描述。简单来说：置信区间95%就是100次里面95次的区间中会包含真实值。

我来举例，然后我们拆解理解。
假设我们班有100个人，测平均身高是170，我现在选的95%的置信区间，每次都选5个人
- 95%置信区间：95%置信区间的意思是，如果重复抽样100次，每次选5个，约95次计算出的区间会"套中"真实的总体参数！
- 我的区间: [168, 172] → 包含170 ✓
- 第1次新区间: [167, 171] → 包含170 ✓ 
- 第2次新区间: [165, 169] → 不包含170 ✗
- 第3次新区间: [175, 179] → 不包含170 ✗
1，2，3... 值不一样是因为平均值是不一样的
