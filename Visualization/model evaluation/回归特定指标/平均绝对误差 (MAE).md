
## definition
本质上是和均方根作用是一致的，都是描述平均误差的大小，但是RMSE的惩罚力度是更大的，因为RMSE有平方

## 计算方式
MAE = Σ|实际值 - 预测值| / n


## 代码实现方式

```
from sklearn.metrics import mean_absolute_error
mae = mean_absolute_error(y_true, y_pred)
```

## MSE，RMSE，MAE对比


| 项目 | MAE | RMSE | MSE |
|------|-----|------|-----|
| 人话含义 | "平均错了多少" | "典型错了多少" | "算法优化用的数" |
| 单位 | 和原数据相同 | 和原数据相同 | 原数据的平方 |
| 对大误差 | 一视同仁 | 重点惩罚 | 严厉惩罚 |
| 直观性 | 最直观 ⭐⭐⭐ | 比较直观 ⭐⭐ | 不直观 ⭐ |
| 计算复杂度 | 最简单 | 中等 | 简单 |


典型 = 考虑了风险的代表值
例如：误差分布: [10, 10, 10, 10, 50]
MAE = 20万   # 简单平均 <br>
RMSE = 24万  # 考虑了那个50万的风险 <br>
