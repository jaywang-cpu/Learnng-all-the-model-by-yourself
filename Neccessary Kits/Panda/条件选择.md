## df.select_dytypes
```
# 数据类型选择列的方法
# 基本语法： df_select_dtypes(include=[''],exclude=['']), include 包含哪些数据类型，exclude相反
# 可选择的类型， int, float64, 字符串，datetime64，bool
# 案例
--------------------1. 选择所有数字列
numeric_cols = df.select_dtypes(include=[np.number])
print("数字列：")
print(numeric_cols)
#    age  height
# 0   25   165.5
# 1   30   175.0
# 2   35   180.2
--------------------2. 选择文本列
text_cols = df.select_dtypes(include=['object'])
print("文本列：")
print(text_cols)
#      name
# 0   Alice
# 1     Bob
# 2 Charlie
```
