# 形象理解

knn就是做菜的方法，可以做出不同的菜最主要的就是四类别菜，都用了knn的方法
1.分类汤（KNeighborsClassifier)
2.回归汤(KNeighborsRegressor)
3.填充汤(KNNImputer)
4.异常检测汤(LocalOutilerFactor)
5.推荐汤（NearestNeighbors）

1.# 配方：找K个最相似的邻居，看他们的类别，投票决定
# 用途：判断邮件是垃圾邮件还是正常邮件
# 原料：邮件特征 → 输出：垃圾/正常

2.# 配方：找K个最相似的邻居，取他们数值的平均
# 用途：预测房价
# 原料：房屋特征 → 输出：具体价格

3.# 配方：找K个最相似的邻居，用他们的值填补空缺
# 用途：补全缺失数据
# 原料：不完整数据 → 输出：完整数据

4.# 配方：看一个样本与邻居的距离，距离太远就是异常
# 用途：发现异常交易
# 原料：交易数据 → 输出：正常/异常

5.# 配方：找到相似用户，推荐他们喜欢的东西
# 用途：电影推荐系统
# 原料：用户偏好 → 输出：推荐列表
